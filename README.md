# Enginerig

**Инженерный анализ операторских ригов** — десктопное приложение для расчёта воспринимаемого веса и дисбаланса съёмочных конструкций (камера, объектив, монопод, точки крепления).  
*Desktop app for camera-rig balance analysis: perceived weight, load on supports and nodes.*

---

## Возможности

- **Схема рига** — точки (противовесы), опора (камера/хват), соединители (монопод, объектив), узлы (быстросъёмы, крепления). Всё рисуется на канвасе и связывается в расчётную модель.
- **Модель расчёта** — фактический вес + ощущаемый (гибрид операторской и сухожильной модели). Дисбаланс в процентах и в граммах.
- **Нагрузка на узлы** — цветовая индикация по степени риска с учётом инерции при быстрой ходбе.
- **Оценка длительности работы** — ориентиры для новичка (до 2 ч) и опытного оператора (до 8 ч).
- **Проекты** — сохранение и загрузка в формате `.rig`, ассоциация файла с программой в Windows.
- **Экспорт** — JPG (FHD) и векторный PDF для чертежей и отчётов.

## Технологии

## Управление

| Действие | Как |
|----------|-----|
| Создать точку | ЛКМ по канвасу |
| Создать соединитель | ЛКМ по точке → ЛКМ по другой точке/опоре |
| Создать узел на соединителе | ЛКМ по соединителю |
| Двигать элементы | Выбрать «Двигать элементы» → перетаскивать |
| Удалить элемент | ПКМ по элементу |
| Изменить длину соединителя | Клик по соединителю → ввод длины → Ctrl+Enter |

Файлы **.rig** можно открывать двойным кликом или перетаскиванием в окно программы.
